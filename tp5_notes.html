tp5安装与配置笔记

1.安装

参考官方文档https//www.kancloud.cn/manual/thinkphp5_1/353946

建议使用composer下载tp5框架: composer create-project topthink/think=5.1.* tp5

有可能报错: zsh: no matches found: topthink/think=5.1.*

原因:没有指定具体的版本

解决方法:composer create-project topthink/think=5.1.31 tp5 # tp5为项目名

或是

一般情况下，composer 安装的是最新的稳定版本，不一定是最新版本，如果你需要安装实时更新的版本（适合学习过程），

可以安装5.1.x-dev版本。

composer create-project topthink/think=5.1.x-dev tp5

下载核心框架

composer update topthink/framework

下载后,在tp5(自己的命名)目录下,终端输入 php think run 会在:8000端口运行框架,出现此图片则为成功

如报错Could not open input file think关闭终端,重新打开.

如报错:

则为端口被占用,可更换其他端口:php think run -p 8080(8080为自己选中的端口)

至此,tp5框架安装完成.

2.配置

1.建议不使用application文件夹,重新新建app文件夹

创建后,拷贝application中所有文件

2.配置tp5/think文件

// [ 应用入口文件 ]

namespace think;

// [ 应用入口文件 ]

namespace think;

// 调试模式开关

define('APP_DEBUG', true);

// 定义CMF根目录,可更改此目录

define('CMF_ROOT', dirname(DIR) . '/');

// 定义CMF数据目录,可更改此目录

define('CMF_DATA', CMF_ROOT . 'data/');

// 定义应用目录

define('APP_PATH', CMF_ROOT . 'application/');

// 定义网站入口目录

define('WEB_ROOT', DIR . '/');

// 加载基础文件

require DIR . '/../thinkphp/base.php';

// 支持事先使用静态方法设置Request对象和Config对象

// 执行应用并响应

Container::get('app', [APP_PATH])->run()->send();

3.至此tp5框架安装完成.默认页面即为:

运行框架:php think run

此链接在浏览器中打开(建议使用chrome,使用F12可进行调试)

出现步骤一页面.

路由访问路径为:application下的文件夹/控制器名/方法名

如路径改为http://127.0.0.1:8000/index/index/hello 便能访问hello方法
